## 🚀 Getting Started

### First Time Setup

1. **Access the Platform**
   - Navigate to your TaaBia installation URL
   - You'll be redirected to the public landing page

2. **Create Your Account**
   - Click "Connexion" in the navigation
   - Choose "Créer un compte"
   - Fill in your details and select your role
   - Verify your email address

3. **Choose Your Role**
   - **Student**: For learning courses and buying products
   - **Instructor**: For creating and selling courses
   - **Vendor**: For selling physical/digital products
   - **Admin**: For platform management (by invitation only)

## 👨‍💼 Admin Guide

### Dashboard Overview

The admin dashboard provides comprehensive platform management with real-time statistics:

- **Total Users**: All registered users across roles
- **Active Courses**: Published courses on the platform
- **Total Revenue**: Platform-wide earnings
- **Recent Orders**: Latest transactions

### User Management

#### Managing Users
1. Navigate to **Users** in the sidebar
2. View all registered users with their roles
3. Use filters to find specific users
4. Click on a user to view/edit their profile

#### User Actions
- **Edit**: Modify user information
- **Toggle Status**: Activate/deactivate accounts
- **Delete**: Remove user accounts (with confirmation)
- **Change Role**: Assign different roles to users

### Course Management

#### Reviewing Courses
1. Go to **Formations** in the sidebar
2. View all courses with their status
3. Click on a course to review details
4. Approve or reject courses

#### Course Actions
- **Publish**: Make courses available to students
- **Archive**: Remove from active listings
- **Edit**: Modify course information
- **Delete**: Remove courses permanently

### Product Management

#### Managing Products
1. Navigate to **Produits** in the sidebar
2. View all vendor products
3. Monitor product status and sales
4. Manage product categories

### Order Management

#### Processing Orders
1. Go to **Commandes** in the sidebar
2. View all platform orders
3. Update order status
4. Process refunds if necessary

### Analytics & Reports

#### Revenue Tracking
- View total platform revenue
- Analyze revenue by time period
- Track commission earnings
- Generate financial reports

#### User Analytics
- Monitor user registration trends
- Track course enrollment rates
- Analyze user engagement
- Generate user reports

## 👨‍🏫 Instructor Guide

### Dashboard Overview

Your instructor dashboard shows:
- **Published Courses**: Number of active courses
- **Total Students**: Students enrolled in your courses
- **Total Earnings**: Revenue from course sales
- **Recent Sales**: Latest course purchases

### Course Creation

#### Creating a New Course
1. Navigate to **Mes cours** in the sidebar
2. Click **Ajouter un cours**
3. Fill in course details:
   - **Title**: Clear, descriptive course name
   - **Description**: Detailed course overview
   - **Price**: Set competitive pricing
   - **Category**: Choose appropriate category
   - **Level**: Beginner, Intermediate, or Advanced
4. Upload course image
5. Click **Publier** to make it live

#### Course Content Management
1. Select a course from your dashboard
2. Go to **Contenu du cours**
3. Add different content types:
   - **Videos**: Upload video files or add YouTube links
   - **PDFs**: Upload course materials
   - **Text**: Write lesson content
   - **Quizzes**: Create interactive assessments

#### Content Organization
- Arrange content in logical order
- Use clear titles for each lesson
- Add descriptions to help students understand
- Include practical exercises

### Student Management

#### Viewing Students
1. Go to **Étudiants** in the sidebar
2. See all students enrolled in your courses
3. View individual student progress
4. Send messages to students

#### Student Progress Tracking
- Monitor completion rates
- Review assignment submissions
- Provide feedback on student work
- Track engagement metrics

### Earnings & Payouts

#### Revenue Tracking
1. Navigate to **Mes Gains** in the sidebar
2. View detailed earnings breakdown
3. Track sales by course
4. Monitor monthly/yearly revenue

#### Requesting Payouts
1. Go to **Paiements** in the sidebar
2. Click **Demander un paiement**
3. Enter payout amount
4. Provide payment details
5. Submit request for processing

### Course Marketing

#### Optimizing Course Listings
- Write compelling descriptions
- Use high-quality images
- Set competitive pricing
- Include preview content

#### Student Engagement
- Respond to student messages promptly
- Provide regular feedback
- Update course content regularly
- Encourage student interaction

## 🔧 Troubleshooting

### Common Issues

#### Login Problems
**Issue**: Can't log in to account
**Solution**:
1. Check email and password spelling
2. Use "Forgot Password" feature
3. Clear browser cache and cookies
4. Try different browser

#### Course Access Issues
**Issue**: Can't access enrolled courses
**Solution**:
1. Verify course enrollment
2. Check payment status
3. Contact course instructor
4. Clear browser cache

#### Payment Problems
**Issue**: Payment not processing
**Solution**:
1. Check payment method details
2. Verify sufficient funds
3. Try alternative payment method
4. Contact support with transaction ID

#### Upload Issues
**Issue**: Can't upload files
**Solution**:
1. Check file size limits
2. Verify file format
3. Ensure stable internet connection
4. Try smaller file size
